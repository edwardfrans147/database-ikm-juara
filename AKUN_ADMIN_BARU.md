Semua password menggunakan kombinasi:
- Huruf besar dan kecil

## Cara Penggunaan

### Login dengan Akun Baru
1. Buka halaman login admin
2. Masukkan username dan password sesuai daftar di atas
3. Gunakan tombol mata untuk melihat password yang diketik
4. Klik Login

### Mengganti Password (Rekomendasi)
Untuk keamanan, disarankan mengganti password default setelah login pertama:
1. Login dengan akun baru
2. Akses menu pengaturan (jika tersedia)
3. Ganti password dengan yang lebih personal

## Troubleshooting

### Login Gagal
- Pastikan username dan password diketik dengan benar (case-sensitive)
- Gunakan tombol mata untuk memverifikasi password
- Periksa koneksi internet
- Refresh halaman dan coba lagi

### Lupa Password
- Hubungi super admin (BidIndustri08#)
- Atau akses langsung ke file `data/admin-users.json` untuk melihat password
