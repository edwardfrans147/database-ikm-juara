# Panduan Penggunaan Export Hasil Penelusuran

## Langkah-langkah Penggunaan

### 1. Melakukan Pencarian Data
1. Buka halaman **"Penelusuran Data"** dari menu sidebar
2. Masukkan salah satu kriteria pencarian:
   - **NIB** (13 digit): Contoh `1234567890123`
   - **NIK** (16 digit): Contoh `3518012345678901`
   - **Nama Lengkap**: Contoh `Ahmad Rizki`
3. Klik tombol **"Cari Data"**
4. Tunggu hasil pencarian muncul

### 2. Export ke Excel (.xlsx)
1. Setelah hasil pencarian muncul, klik tombol **"Export Excel"**
2. File akan otomatis terdownload dengan nama:
   ```
   Penelusuran_[Nama_Lengkap]_[Tanggal].xlsx
   ```
3. Buka file dengan Microsoft Excel atau aplikasi spreadsheet lainnya
4. File berisi:
   - Data dasar IKM Binaan
   - Semua layanan yang pernah diikuti
   - Riwayat pelatihan
   - Ringkasan total layanan dan pelatihan

### 3. Export ke PDF
1. Setelah hasil pencarian muncul, klik tombol **"Export PDF"**
2. PDF akan terbuka di tab/window browser baru
3. Untuk menyimpan atau print:
   - Klik tombol **"Print / Save as PDF"** di halaman
   - Atau gunakan Ctrl+P (Windows) / Cmd+P (Mac)
   - Pilih "Save as PDF" sebagai printer destination

### 4. Mencari Data Lainnya
1. Klik tombol **"Cari Data Lainnya"**
2. Form pencarian akan ter-reset otomatis
3. Hasil pencarian sebelumnya akan hilang
4. Cursor akan fokus ke field NIB
5. Halaman akan scroll ke form pencarian
